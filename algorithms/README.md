# Algorithms

This folder contains algorithm implementations from the Data Structures and Algorithms Specialization course.

## Structure

- **`solutions/`** - Python implementation files organized by course module
- **`notebooks/`** - Jupyter notebooks for documentation and experimentation

## Modules

### Module 1 - Algorithmic Warmup
Basic algorithmic problems and complexity analysis
- Maximum Pairwise Product

### Module 3 - Greedy Algorithms
Locally optimal choice algorithms
- Car Fueling
- Covering Segments
- Different Summands
- Fractional Knapsack
- Largest Number
- Maximum Dot Product
- Money Change (Greedy)

### Module 4 - Divide and Conquer Algorithms
Recursive problem-solving techniques
- Binary Search
- Closest Points
- Counting Inversions
- Majority Element
- Merge Sort
- Points and Segments
- Randomized Quick Sort

### Module 5 - Dynamic Programming
Optimal substructure and overlapping subproblems
- Edit Distance
- Knapsack (0/1)
- Longest Common Subsequence
- Money Change (DP)
- Placing Parentheses
- Primitive Calculator

## Usage

Navigate to any module directory and run algorithms with standard input:
```bash
cd "solutions/module 5 - Dynamic programming"
python3 money_change_dp.py < input.txt
```

## Notebooks

The `notebooks/` folder contains:
- **DivdeAndConquer.ipynb** - Interactive examples for divide and conquer algorithms
- **DynamicProgramming.ipynb** - Detailed analysis of DP algorithms
- **Python.ipynb** - General Python programming reference
