# Data Structures

This folder contains data structure implementations and analyses from the Data Structures and Algorithms Specialization course.

## Structure

- **`solutions/`** - Python implementation files for data structure problems
- **`notebooks/`** - Jupyter notebooks with detailed analysis and visualizations

## Module 1 - Basic Data Structures

This module covers fundamental data structures and their applications:

### Topics Covered

- **Stacks** - LIFO (Last In, First Out) data structure
- **Queues** - FIFO (First In, First Out) data structure  
- **Trees** - Hierarchical data structures
- **Hash Tables** - Fast lookup data structures

### Implementation Files

Located in `solutions/Data structures module 1 - basic data structures/`

### Interactive Notebooks

The `notebooks/` folder contains detailed Jupyter notebooks:

- **brackets_in_code.ipynb** - Stack-based bracket matching algorithm
- **compute_tree_height.ipynb** - Tree traversal and height computation
- **extending_stack_interface.ipynb** - Advanced stack operations
- **maximum_sliding_window.ipynb** - Deque-based sliding window maximum
- **network_packaging_problem.ipynb** - Queue-based network simulation

## Usage

Each notebook provides:
- Step-by-step algorithm explanations
- Interactive code examples
- Visualization of data structure behavior
- Time and space complexity analysis
- Comparison between different approaches

## Key Learning Objectives

- Understanding when to use different data structures
- Analyzing time and space complexity
- Implementing efficient algorithms using appropriate data structures
- Solving real-world problems with fundamental data structures
